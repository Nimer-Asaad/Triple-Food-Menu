# Triple Food Menu Display

A simple online menu display website for **Triple Food Restaurant**.

The website allows the restaurant owner to upload, delete, and reorder menu images from a private admin dashboard. The public page displays the menu images in a clean vertical layout and includes direct contact links for customers.

Live Website:  
https://triple-food-menu.vercel.app/

---

## Features

- Public menu display page
- Admin dashboard for managing menu images
- Upload menu images
- Delete images
- Reorder images using Move Up / Move Down
- Cloudinary image storage
- MongoDB Atlas database
- Password-protected admin page
- Admin session timeout
- Direct contact links:
  - Instagram
  - Facebook
  - WhatsApp
  - Phone call
